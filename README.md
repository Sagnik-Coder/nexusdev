## 🌌 About The Project

**NEXUSDEV** is an ultra-modern, glassmorphism-inspired web simulation application designed for next-generation software engineers and financial quantitative operators. Moving far beyond passive, legacy video-based learning frameworks, NexusDev introduces an **anti-subscription micro-transaction paradigm** allowing users to unlock specific modular lectures for exactly **₹125 per topic** via simulated UPI processing rails.

The system dynamically branches into two separate functional sandboxes: a production-grade **Programming Lab** and a data-dense **Trading Risk Simulator**.

### ✨ Key Features

- **Multi-Track Core Switcher:** Seamlessly shift the entire app architecture between an AI-Native software engineering IDE and a financial candlestick market graph environment.
- **Micro-Modular Ledger Engine:** Gated asset security layer that monitors access tokens, launching an automated Razorpay/UPI gateway simulation to unlock premium content blocks for ₹125.
- **Embedded Monaco Workspace:** Features a fully interactive code compiler suite inside the browser, complete with line diagnostics and terminal stream outputs.
- **Financial Charting Matrix:** Interactive paper-trading canvas built with responsive trend arrays, spot metrics, and volatility data streams.
- **Context-Aware Neural Advisors:** Real-time AI mentors (*Alex the Systems Architect* & *Marcus the Risk Advisor*) who dynamically switch personalities and prompt parameters depending on your active track.
- **Optimized Cybernetic UI:** Premium dark-mode layout leveraging smooth web viewport keyframe animations, sleek glassmorphic overlays (`backdrop-blur`), and balanced rendering states.

## 🛠️ Tech Stack

This project is compiled and bundled with high-performance modern web technologies:

* **Core Framework:** [React.js](https://react.dev/) (Component-driven state engine)
* **Code Editor Base:** [@monaco-editor/react](https://www.npmjs.com/package/@monaco-editor/react) (The underlying workspace power of VS Code)
* **Styling Engine:** [Tailwind CSS](https://tailwindcss.com/) (Utility-first atomic styling layouts)

* **Build System:** Webpack via Create React App configuration pipeline
* **Deployment System:** [GitHub Pages](https://pages.github.com/) (Automated secure delivery edge)

## 🚀 Local Deployment Sequence

To run a localized preview instance of this platform matrix directly from your machine or cloud workspace terminal, initialize these sequential phases.

### Setup Configuration

1. Clone the repository into your environment:
   ```
   git clone [https://github.com/Sagnik-Coder/nexusdev.git](https://github.com/Sagnik-Coder/nexusdev.git)
   ```

2. Navigate into the core project workspace:
  ```
cd nexusdev
  ```


3. Install all deep dependencies outlined inside the project configuration ledger:
```
npm install
```



### Execution Controls

* Run the local development server with active HMR (Hot Module Replacement):
```
npm start

```


* Compile the highly optimized production build and push it straight to your live GitHub Pages domain:
```
npm run deploy

```
